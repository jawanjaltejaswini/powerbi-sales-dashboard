# Power BI Sales Dashboard

## Project Overview
This project analyzes sales data using Power BI.

## Tools Used
- Power BI Desktop
- Excel

## Dashboard Features
- Total Sales Card
- Total Profit Card
- Sales by Product (Bar Chart)
- Sales by Category (Pie Chart)
- Profit by Region (Column Chart)
- Sum of Sales by Region (Line Chart)
- Sum of Profit by Region (Clustered Chart)
- Region and Category Slicers

## Dataset Columns
- Order_ID
- Product
- Category
- Region
- Sales
- Profit
